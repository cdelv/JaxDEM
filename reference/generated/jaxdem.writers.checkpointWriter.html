jaxdem.writers.checkpointWriter#

Interface for defining data writers.

This module provides a high-level VTKWriter frontend, a VTKBaseWriter plugin interface, and concrete writers (e.g., SpheresWriter, DomainWriter) for exporting JAX-based simulation snapshots to VTK files. It also contains utilities for safe directory cleanup and an (experimental) checkpointing wrapper.
